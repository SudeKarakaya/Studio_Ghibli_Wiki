# Studio Ghibli Wiki

A React-based web application that explores the Studio Ghibli universe.

This project allows users to browse Studio Ghibli films, characters, and locations using data fetched from the public Ghibli API.  
The application is built with React, Vite, and Tailwind CSS, focusing on clean UI design and responsive layout.

## Features
- Browse Studio Ghibli films with detailed information
- View characters and see which films they appear in
- Explore locations from the Ghibli universe
- Search functionality
- Load more pagination for characters
- Random film discovery feature

## Technologies Used
- React
- Vite
- React Router
- Tailwind CSS
- Fetch API
